# ✅ Method 1: Manual Conversion (Fastest)

**Transform the URL using this rule:**

Change github.com → raw.githubusercontent.com

Remove /blob

**Keep the rest the same**

So this:
`https://github.com/7426-Pair-Of-Dice/2026RobotPics/blob/main/team_number/picture.png`

Becomes:
`https://raw.githubusercontent.com/7426-Pair-Of-Dice/2026RobotPics/main/team_number/picture.png`

This is the exact format GitHub recommends.

# ✅ Method 2: Add ?raw=true

**GitHub also supports a shortcut:**

`https://github.com/7426-Pair-Of-Dice/2026RobotPics/blob/main/images/picture.png?raw=true`

GitHub will redirect this to the proper raw link.
